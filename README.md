Yoldoro - Egg Focus Timer
Yoldoro adalah aplikasi produktivitas berbasis web dengan tema telur yang lucu. Dibuat untuk membantu manajemen waktu menggunakan teknik Pomodoro (Fokus & Rehat).

Link Website: https://yoldoro.web.app

Fitur Utama:
- Timer Pomodoro yang bisa diatur sendiri durasinya.
- Target Belajar untuk memantau total waktu produktif.
- Pemutar musik Lofi dari SoundCloud.
- Fitur tambah lagu favorit secara custom.
- UI yang clean dan responsif.

Teknologi yang Digunakan:
- HTML5, CSS3, dan JavaScript Vanilla.
- Google Firebase Hosting untuk deployment.
- SoundCloud API untuk integrasi musik.
